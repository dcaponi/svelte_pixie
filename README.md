# Svelte Pixie
A demo of the PKCE Auth Flow using Svelte

## Get Started
1. Clone Me
2. `npm i`
3. Add a `.env` file with `ONELOGIN_CLIENT_ID=<oidc_app_client_id>` where the client_id comes from a OneLogin OIDC App
4. npm run dev
